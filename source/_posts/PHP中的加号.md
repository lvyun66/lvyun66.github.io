PHP算术运算符很常见，例如1+2、2.0+3.0等等，但在PHP中加号（+）不仅仅可以在数字之间相加，还可以在别的类型之间进行运算。加号运算符（+）都是在哪些类型之间运算呢？下面让我们一一列举。

<!-- more -->

## 数字
```php
$foo = 1;
$bar = 2;
echo $foo + $bar;

// output
// 3
```

## 数组
PHP的加号运算符还可以在数组之间应用，例如：
```php
<?php

public function add($a, $b)
{
    $a += $b;
    var_dump($a);
}

add([1,2], [3,4,5]);
// output
// array(3) {
//   [0]=>
//   int(1)
//   [1]=>
//   int(2)
//   [2]=>
//   int(5)
// }
```

数组之间相加，前面的数组会把键值相同后面的数组覆盖。如果前面的数组的键不在后面的数组中，那么将会丢着这部分。
